# chessbook
Chess.com GUI and Chess.com historical game script

Goals
1. Gamify Naroditsky's speedruns for the average chess player
  a. show how many times you've played this openining.
  b. categorize openings at following levels
    i. 0 - 1000
    ii. 1000 - 1100
    iii. 1100 - 1200
    iv. 1200 - 1300
    v. 1300 - 1400
    vi. 1400 - 1500
    vii. 1500 - 1600
    viii. 1600 - 1700
    ix. 1700 - 1800
    x. 1800 - 1900
    xi. 1900 - 2000
