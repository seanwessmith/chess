# chessbook
Chess.com GUI and Chess.com historical game script

Goals
1. Gamify Naroditsky's speed runs for the average chess player
2. show how many times you've played this opening.
3. categorize openings at the following levels

1. 0 - 1000
2. 1000 - 1100
3. 1100 - 1200
4. 1200 - 1300
5. 1300 - 1400
6. 1400 - 1500
7. 1500 - 1600
8. 1600 - 1700
9. 1700 - 1800
10. 1800 - 1900
11. 1900 - 2000
